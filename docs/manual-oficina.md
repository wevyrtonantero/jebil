# Manual da Oficina

## Fluxo principal

1. Entrar com usuario `OFICINA`.
2. Abrir a tela `Oficina`.
3. Escolher um atendimento da fila.
4. Selecionar o mecanico responsavel.
5. Assumir o atendimento.
6. Atualizar status conforme o andamento.
7. Retornar para a fila quando necessario.
8. Concluir o servico.
9. Se o sistema indicar pagamento confirmado, liberar retirada.
10. Se o sistema indicar pagamento pendente, orientar o cliente a procurar a recepcao.

## Cuidados

* O perfil `OFICINA` nao pode cancelar atendimento.
* O perfil `OFICINA` nao confirma pagamento.
* O painel interno nao substitui a tela operacional.
