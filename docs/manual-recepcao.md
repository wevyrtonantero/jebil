# Manual da Recepcao

## Fluxo principal

1. Entrar com usuario `ADMIN` ou `RECEPCAO`.
2. Cadastrar ou localizar o cliente.
3. Cadastrar ou localizar a motocicleta do cliente.
4. Abrir o atendimento em `Recepcao`.
5. Acompanhar o status em `Atendimentos`.
6. Confirmar pagamento quando houver quitacao.
7. Liberar retirada quando o servico estiver concluido e pago.
8. Confirmar retirada para finalizar a OS.

## Cuidados

* Nao criar cliente duplicado quando o sistema sugerir reativacao.
* Nao criar motocicleta duplicada quando o sistema sugerir reativacao.
* Nao liberar retirada com pagamento pendente.
* Consultar historico quando houver divergencia operacional.
