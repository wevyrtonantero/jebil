# Parte 15 - Tela da Oficina

## Objetivo

Implementar a visualizacao da fila e os cards operacionais da oficina.

## Tela implementada

* `frontend/src/pages/OficinaAdminPage.jsx`

## Regras implementadas

* lista da fila de atendimentos aguardando
* selecao de mecanico para assumir atendimento
* cards por mecanico ativo
* troca de status operacional
* retorno para fila
* conclusao de servico com alerta de pagamento pendente

## Testes executados

### Sucesso

* build do frontend aprovado
* lint do frontend aprovado

## Limitacao atual

O uso completo da tela depende da API com dados reais de mecanicos e atendimentos.

## Comandos

```bash
cd frontend
npm run lint
npm run build
```
